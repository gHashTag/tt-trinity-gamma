# Trained gf16 model + eval set (sklearn digits)

Reusable artifact for the Defect-1 (gf16_mul) task-impact study -- the "external
lever" beyond the synthetic proxy, materialized so it can be replayed without
retraining. All values are gf16 codes [S(1) E(6) M(9)], bias 31.

## Architecture
2-layer MLP: 15 PCA features -> 15 ReLU hidden -> 10 classes (logistic output).
Each layer is one 16-element `dot16` (15 weights + 1 bias column; inputs carry a
trailing bias term = gf16 1.0 = 0x3e00). ReLU between layers (negative gf16
code -> 0). Trained on sklearn `digits` (8x8, 10 classes), seed 7.

## Files
- `W1.mem`   15 hidden units x 16 gf16 weight codes
- `W2.mem`   10 classes x 16 gf16 weight codes
- `eval_X.mem`  450 test inputs x 16 gf16 codes (15 PCA + bias)
- `eval_y.txt`  450 true labels

## Provenance
float (sklearn) test accuracy: 95.11%
Generated by `test/gen_gf16_model.py` (deterministic). Replay through the actual
RTL with `test/run_trained_gf16_eval.py` (no sklearn needed) -> reports shipped vs
corrected (gf16_v2) accuracy + the Defect-1 argmax-flip number.
